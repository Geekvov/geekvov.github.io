A 记录
使用场景: 添加 A 记录可实现将域名指向 IP 地址
主机记录：www.example.com，则主机记录为：www；（example.com，主机记录为：@）
记录值：IP 地址，填写 IPv4 地址
TTL：为缓存时间，数值越小，修改记录各地生效时间越快。

CNAME 记录
主机记录：www
记录值：username.github.io
同时还需要再 GitHub 仓库中Settings，点击 Pages，在Custom domain 设置成自己够买的域名。此时博客仓库会自动生成 CNAME (无后缀名)，里面的内容是你的域名。
