本人发布博客的一般流程：
(1) 在你使用 git 工具搭建 hexo 博客的过程中，会生成很多文件夹，需要找到此文件hexo\source_posts，在此处新建 “.txt” 文件，把此文件改为 “写博客.md”（此处仅仅是举个栗子🌰）

(2) 使用 Typora 软件编辑内容：
正常编辑完再上传到博客上，是不会显示标题的。需要按照下图格式对标题进行修改，先在 Typora 顶部输入 “title: 如何写博客”，再用鼠标选中此文本内容，输入快捷键 “ctrl + /” （打开源代码模式），在此标题的开头与结尾输入 “—”，再正常编辑正文内容，保存，就可以了。

（3）每次 “.md” 文件编辑完成后，在hexo根目录下，鼠标右键“Git Bash Here”，依次输入:
$ hexo clean
$ hexo g
$ hexo s
$ hexo d
